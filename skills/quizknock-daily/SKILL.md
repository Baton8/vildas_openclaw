# 今日のQuizKnock - Daily Post Skill

毎日、QuizKnockの最新活動を起点にしたX向け日記投稿を生成して、指定スレッドに投稿するスキル。

## 投稿先

- **Channel:** C0AFYPKLTK5 (`#vil_botan_lab`)
- **Thread timestamp:** 1771641971.913779（このスレッドに毎日注ぎ足す）

## 実行タイミング

毎朝 8:00 JST ごろ（cronで自動実行）

## 重複防止

`skills/quizknock-daily/log/YYYY-MM-DD.md` が存在する場合はスキップ。

## 手順

### 1. 最新情報収集（web_search × 2〜3回）

```
① web_search("QuizKnock 新着動画 OR 最新情報 YYYY年M月")
② web_search("QuizKnock site:x.com OR YouTube YYYY年M月DD日")
③ 直近の動画タイトル・テーマを特定する
```

### 2. トリビア掘り下げ

- 動画のテーマから「知ってた？」と言いたくなる豆知識を調査
- `web_search("[テーマ] 歴史 豆知識 トリビア")`
- QKらしい「知の角度」で切り取る（意外な起源・数字・対比など）

### 3. 投稿フォーマット（Slack用 → Xに転載想定）

投稿は以下の4ブロック構成。Slackへは1メッセージにまとめて投稿する。

```
---
📅 今日のQuizKnock | M/DD（曜）

[QK最新動画・ニュースへの一言コメント]
[「ところで…」でトリビアへ接続]

#QuizKnock #クイズノック #今日のQuizKnock
---
📌 [トリビア見出し]

[2〜3文の豆知識。意外な数字・歴史・対比を入れる]
---
❓ 今日のクイズ

[豆知識から派生した4択クイズ]
A）〜　B）〜　C）〜　D）〜

答えは↓
---
正解は [X）〜〜〜] 🎉

[答えの補足 + QKと絡めた一言で締める]
```

### 4. ログ保存

`skills/quizknock-daily/log/YYYY-MM-DD.md` に投稿内容を保存する。

### 5. git push

変更をコミット・プッシュする。

## トーン

- 親しみやすく、でも豆知識は正確に
- 「QK起点 → 豆知識 → クイズ」の一本道を崩さない
- クイズの答えは必ず正しいものを使う（不確かな場合は別テーマを選ぶ）
- 絵文字は控えめに（📅 📌 ❓ 🎉 程度）
