# Quiz Host

クイズを出題して解答を受け付けるスキル。作問の過程はユーザーに見せない。

## 使い方

1. テーマを受け取る（指定がなければ自由なテーマを選ぶ）
2. 形式を確認する（早押し or 4択。指定がなければ早押し）
3. 「調査中です〜！」とチャットに一言報告してから `research` スキルでテーマを調査する（**調査内容の詳細はチャットに出力しない**）
4. 「作問中です〜！」とチャットに一言報告してから `quiz-create` スキルで問題を作成する（**作問過程の詳細はチャットに出力しない**）
5. 完成した問題文だけを出題する

## 出題フォーマット

**早押しクイズ:**
```
🎯 問題です！

[問題文]

（わかったら答えてください！）
```

**4択クイズ:**
```
🎯 問題です！

[問題文]
① ...
② ...
③ ...
④ ...

（番号で答えてください！）
```

## 解答受付

- ユーザーが回答したら、正解 or 不正解を伝える
- 正解の場合：「正解です！🎉」＋簡単な解説
- 不正解の場合：「惜しい！」または「残念！」＋正解と解説
- 「降参」「わからない」「答えは？」などのメッセージは不正解扱いで正解を教える

## 重要なルール

- **作問の思考過程・検索ログ・審査コメントは一切チャットに出力しない**
- 問題文と解答受付のやり取りのみ表示する
- 解答が出るまで正解を絶対に明かさない
